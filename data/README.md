# 데모용 데이터셋

## 개요
이커머스 AI 에이전트 데모를 위한 한국어 번역 데이터셋입니다.
Amazon 상품/리뷰 데이터를 기반으로 자연스러운 한국어 쇼핑몰 리뷰 형태로 가공했습니다.

## 파일 구성

### 1. `products_ko.csv` (47개 상품)
25개 브랜드의 다양한 상품 (의류 중심: 폴로셔츠, 청바지, 팬츠, 쇼츠 등)

| 컬럼 | 설명 |
|------|------|
| `asin` | Amazon 상품 고유 ID |
| `title` | 원본 영문 상품명 |
| `title_ko` | 한국어 번역 상품명 |
| `brand_name` | 브랜드명 (원본 유지) |
| `price_value` | 가격 (USD) |
| `breadcrumbs` | 원본 카테고리 경로 |
| `breadcrumbs_ko` | 한국어 카테고리 경로 |
| `all_images` | 상품 이미지 URL 목록 |
| `rating_stars` | 평균 별점 |
| `about_item` | 상품 설명 |
| `rating_count` | 총 리뷰 수 |

### 2. `reviews_ko.csv` (120건 리뷰)
별점별 균등 배분된 리뷰 데이터

| 컬럼 | 설명 |
|------|------|
| `productASIN` | 상품 ASIN (products_ko.csv의 asin과 매칭) |
| `rating` | 별점 (1~5) |
| `reviewText` | 원본 영문 리뷰 |
| `reviewTitle` | 원본 영문 리뷰 제목 |
| `reviewTitle_ko` | 한국어 리뷰 제목 |
| `reviewText_ko` | 한국어 리뷰 본문 |
| `sentiment_score` | 감성 점수 |
| `verifiedPurchase` | 인증 구매 여부 |
| `review_type` | 리뷰 온도감 분류 |
| `reviewID` | 리뷰 고유 ID |

#### 리뷰 분포
| review_type | 건수 | 별점 | 설명 |
|-------------|------|------|------|
| 격노 | 20건 | 1점 | 강한 불만, 환불/반품 요청 |
| 불만 | 15건 | 2점 | 실망, 기대 이하 |
| 보통 | 20건 | 3점 | 중립적, 애매한 평가 |
| 만족 | 25건 | 4점 | 전반적 만족, 부분적 아쉬움 |
| 극찬 | 25건 | 5점 | 강한 만족, 재구매 의사 |
| 문의 | 15건 | 혼합 | 질문형 리뷰 (사이즈, 소재, 세탁 등) |

### 3. `brand_tone_presets.json` (3개 프리셋)
운영자가 선택할 수 있는 브랜드 대댓글 톤 가이드

| 프리셋 ID | 이름 | 특징 |
|-----------|------|------|
| `friendly` | 친근한 톤 | 캐주얼 브랜드, 따뜻한 존댓말, 적절한 이모지 |
| `professional` | 전문적인 톤 | 프리미엄 브랜드, 격식 있는 표현, 정확한 정보 |
| `emotional` | 감성적인 톤 | 감성 브랜드, 깊은 공감, 진심 어린 표현 |

각 프리셋에 포함된 항목:
- 톤 설명 (`description`)
- 톤 키워드 5개 (`keywords`)
- 긍정/부정/문의 대댓글 샘플 (`sample_replies`)

## 참고사항
- CSV 인코딩: UTF-8 (BOM 포함)
- 한국어 번역은 자연스러운 쇼핑몰 리뷰 말투로 작성 (직역 아님)
- 1~2점 리뷰는 원본 데이터에 부족하여 일부 목업 데이터 포함 (`reviewID`가 `MOCK_`으로 시작)
- 브랜드명은 원본 유지
- 생성일: 2026-04-22
